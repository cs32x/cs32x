# cs32x

Welcome. We're glad you're here and interested in our course.

### Get started

On this page, click the green "Use this template" button to create a `cs32x` repository under your GitHub account. In the window that pops up, do the following:

1.  Set the new repository's name to `cs32x`
2.  Set this repository's visibility to "Private"
3.  All other settings are fine

When you click the green "Create repository" button at the bottom of the page, you’ll be taken to the new repository in your GitHub account. On this page:

*  Click the green “<> Code” button
*  Select the "Codespaces" tab
*  Click the green "Create codespace on main" button

 This will launch a codespace, and you can now start following the directions in the online CS32x course.

### What's in this starter repo

Configuration files adapt the launched codespace and VS Code for the Web in the following ways:

*  Installs a Python programming environment
*  Upgrades the Python package manager `pip`
*  Silences as much as possible hints and AI suggestions in VS Code

You will learn to use AI during coding, but we don't start you in an environment where the AI is constantly flashing incomprehensible stuff at you as you're trying to learn the foundations of computational thinking and computer programming.

The utility `grab32.py`, which you'll use to grab the files you need as you start each unit in the course.
